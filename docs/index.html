Sconce

Drag PNG images to placeholders.

There is no lack of PNG -> ICO converters on the interwebs, but this was more of a challenge to myself, and an homage to Apple's Icon Composer.

ICO

For PNG based ICOs the format is fairly simple.

  • 6 Byte header (ICONDIR)
  • Followed by a 16 Byte image entry (ICONDIRENTRY) for each image
  • Then each image's full binary data is appended

Sconce

This is a PWA. It can be used offline, it does not communicate with any servers. All images and icons stay on your machine, everything is done in your browser. Source code is available on GitHub.
